Тестовое задание по теме Document Object Model

Задание №1

Создать модуль, который экспортирует функцию `prepend` prepend имеет два параметра, в которые нужно передать элементы Задача функции - вставить второй элемент в начало первого. Например: `prepend(container, newElement)` - newElement должен быть добавлен в начало элемента container.

Задание №2

Создать модуль, который экспортирует функцию `deleteTextNodes>` Эта функция принимает на вход элемент и должна удалить все текстовые узлы внутри указанного элемента. Функция может работать не рекурсивно, то есть не заходить внутрь дочерних элементов контейнера.

Задание №3

Реалзиовать функцию, описанную в ДЗ-2, рекурсивно

Задание №4

Создать модуль, который экспортирует функцию `scanDOM`. `scanDOM` должна перебирать все узлы на странице и выводить в консоль статистику по элементам и классам на странице. Например:

  • Тэгов div: 10
  • Тэгов a: 5
  • Текстовых узлов: 100
  • Элементов с классом c1: 10
  • Элементов с классом c2: 20
